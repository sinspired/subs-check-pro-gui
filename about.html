Subs-Check⁺ PRO GUI

高并发网络代理质量检测桌面终端

前台：Wails v3 GUI

  • Windows / macOS / Linux 全平台一致体验
  • 系统托盘与原生窗口控件无缝集成
  • 侘寂风设计，摒弃多余视觉干扰
  • 零配置启动，可视化免命令操作

后台：Subs-Check Pro 核心

  • 高并发自适应流水线，测活测速解锁
  • WebUI与终端双引擎配置分析

  • 极致的内存控制(百万量级低开销)
